# Golf Olympics Score Tracker — プロジェクト引き継ぎ

## プロジェクト概要

ゴルフの「オリンピック（マイホール）」ゲーム専用のスコア管理アプリ。
SwiftUI（iOS）での本実装を目標とし、現在はReact（JSX）でプロトタイプを作成済み。

---

## ゲームルール

- **形式**：マイホール（ホールごとに独立した勝負）
- **人数**：2〜4人（可変）
- **ホール数**：18ホール
- **メダル割り当て**：全員グリーンオン後、ピンから遠い順に金・銀・銅・鉄を割り当て
- **得点条件**：そのホールで **1パット（ワンパット）で沈めた** プレイヤーのみ点数を獲得
- **点数配分（固定）**：

| メダル | 点数 |
|--------|------|
| 🥇 金  | 4pt  |
| 🥈 銀  | 3pt  |
| 🥉 銅  | 2pt  |
| 🔩 鉄  | 1pt  |

- 2人プレイ → 金・銀のみ
- 3人プレイ → 金・銀・銅のみ
- 4人プレイ → 金・銀・銅・鉄すべて

### スペシャルオプション

ホールごとに以下の特殊ポイントを付与できる。

| オプション | 点数 | 条件 |
|-----------|------|------|
| 💎 ダイヤ  | 5pt  | メダルの代わりに選択。メダル点は無効になる |
| 🚩 竿イチ権 | +3pt | メダルを持つプレイヤーへの加算ボーナス |
| 📍 ニアピン | 2pt  | ショートホール限定。1人のみ選択可能 |

- **ダイヤ**：ホールインワン等の特別プレーに使用。選択するとメダル選択が無効化される
- **竿イチ権**：ピンに最も近い位置にボールが止まった（竿一本以内）ときのボーナス。メダルとの重複可
- **ニアピン**：ショートホール（★ ショートホールON）のとき有効。グリーン上でピンに最も近いプレイヤーが獲得

## 画面構成

```
[1. スタート画面]
  - プレイヤー人数選択（2〜4人）
  - プレイヤー名入力
  - ゲーム開始ボタン

[2. ホール入力画面]
  - 現在ホール番号 + プログレスバー
  - 各プレイヤーへのメダル選択ボタン（ワンパット成功時のみ選択）
  - 再タップで選択解除
  - スペシャルオプション：💎 ダイヤ / 🚩 竿イチ権（各プレイヤー行に表示）
  - ★ ショートホールON → 📍 ニアピン選択エリアが展開
  - ホール遷移時は選択状態をリセット（保存済みデータは復元）
  - ミニスコアボード（リアルタイム合計表示）
  - 「戻る」「次のホールへ／結果を見る」ボタン
  - ✕ ゲームを中止してトップへ戻るボタン（確認ステップあり）

[3. 結果画面]
  - 最終順位・合計ポイント表示
  - ホール別スコア詳細（折りたたみ）
  - 💴 レート計算・集計ボタン（レートを自由入力 → 精算額を自動計算）
  - ✏️ スコアを修正するボタン（最終ホールに戻り、←戻るで任意ホールまで遡れる）
  - 🔄 新しいゲームを始めるボタン（スタート画面に戻る＆データリセット）
```

## データモデル（SwiftUI実装用）

```swift
// プレイヤー
struct Player: Identifiable {
    let id: UUID
    var name: String
}

// メダルの種類と点数
enum Medal: Int {
    case gold   = 4  // 🥇
    case silver = 3  // 🥈
    case bronze = 2  // 🥉
    case iron   = 1  // 🔩
}

// スペシャルオプション
enum Special {
    case diamond  // 💎 5pt（メダルの代わり）
    case saoichi  // 🚩 +3pt ボーナス（メダルに加算）
    case neapin   // 📍 2pt（ショートホール限定、1人のみ）
}

// 1ホールの結果
struct HoleResult {
    let holeNumber: Int           // 1〜18
    var medals: [UUID: Medal]     // playerID → 獲得メダル
    var diamonds: Set<UUID>       // ダイヤ選択プレイヤー（メダル無効）
    var saoichi: Set<UUID>        // 竿イチ権プレイヤー（メダルに+3pt）
    var neapin: UUID?             // ニアピン獲得プレイヤー（1人のみ）
    var isShort: Bool             // ショートホールフラグ
}

// ゲーム全体の状態
class GameSession: ObservableObject {
    var players: [Player]
    var holeResults: [HoleResult]  // 最大18要素
    var currentHole: Int

    func totalScore(for playerID: UUID) -> Int
}
```

## プロトタイプ（React）のセットアップ手順

```bash
# 1. Viteプロジェクト作成
npm create vite@latest golf-olympics -- --template react

# 2. 移動＆インストール
cd golf-olympics
npm install

# 3. プロトタイプファイルを配置
cp GolfOlympics.jsx src/App.jsx

# 4. 起動
npm run dev
# → http://localhost:5173 で動作確認
```

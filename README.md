# Golf Olympian ⛳

ゴルフの「オリンピック（マイホール）」ゲーム専用のスコア管理アプリ。

## ゲームルール

- **形式**：マイホール（ホールごとに独立した勝負）
- **人数**：2〜4人
- **ホール数**：18ホール

全員グリーンオン後、ピンから遠い順に金・銀・銅・鉄のメダルを割り当て、そのホールで **ワンパット（1パット）** で沈めたプレイヤーのみ点数を獲得。

| メダル | 点数 |
|--------|------|
| 🥇 金  | 4pt  |
| 🥈 銀  | 3pt  |
| 🥉 銅  | 2pt  |
| 🔩 鉄  | 1pt  |

人数に応じてメダル数が変わります（2人→金銀、3人→金銀銅、4人→金銀銅鉄）。

## 機能

- プレイヤー設定（2〜4人、名前入力）
- ホールごとのメダル選択（排他制御、再タップで解除）
- ミニスコアボード（リアルタイム合計表示）
- 最終結果画面（順位・合計ポイント）
- ホール別スコア詳細
- レート計算・精算（任意レートで支払い額を自動計算）
- スコア修正（任意ホールまで遡って編集）

## セットアップ

```bash
npm install
npm run dev
# → http://localhost:5173/golf-olympian/ で起動
```

## ビルド

```bash
npm run build   # dist/ に出力
npm run preview # ビルド結果を確認
```

## デプロイ

`main` ブランチへのプッシュで GitHub Actions が自動的に GitHub Pages へデプロイします。
